Dataset Information

The datasets used in the Path4Med project include water and soil quality measurements collected from the Tejo Vulnerable Zone in Portugal, specifically at Quinta da Cholda, Golegã. The data is sourced from the Sistema Nacional de Informação de Recursos Hídricos (SNIRH) and includes parameters such as nitrate levels, pH, and other relevant metrics for monitoring pollution in the Mediterranean Agro-Hydro-System.

Database Centralization

To manage and analyze the collected data efficiently, we centralized all datasets into a relational database using DBeaver, a universal database management tool. DBeaver allowed us to create a structured database schema, import the CSV files, and perform queries to support the Path4Med project's analysis and reporting needs.

Visualization and Mapping in QGIS

The Path4Med project utilized QGIS to visualize and map the monitoring points across Portugal Continental. The map highlights the Tejo Vulnerable Zone, with specific points representing monitoring stations at Quinta da Cholda, Golegã. Each point is color-coded based on pollution levels, as indicated in the legend.
